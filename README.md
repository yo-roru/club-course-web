# club-course-web

網頁開發基礎課程教學教材與範例程式碼。

本專案為大三於駭客社擔任網頁設計講師時所設計之基礎課程內容，包含前端與後端兩堂實作導向課程。

---

## 📚 課程內容

### 1️⃣ Front-end（第一堂課）
基礎網頁設計與前端開發概念：
- HTML
- CSS
- JavaScript

重點說明：
- 網頁基本結構
- 樣式設計與版面配置
- 基礎互動功能實作

---

### 2️⃣ Back-end（第二堂課）
Linux 環境建置與基礎後端開發：

- Ubuntu 環境架設
- Web Server 建置（Apache）
- Database 建置（MariaDB）
- PHP 基礎語法
- SQL 基礎操作
- 前後端整合概念
